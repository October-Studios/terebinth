## ðŸ”° Getting Started
### Installation
The terebinth interpreter can be installed using cargo:
```shell
cargo install terebinth
```
or it can be built from source:
```shell
git clone git@github.com:October-Studios/terebinth.git
cd terebinth
cargo build --release
```
